MathLive plugin

Basic usage

This plugin allows user to use MathLive to edit math formulas.

You can try to insert some formulas

Some formulas to copy/paste to try it

The code


// MathJax inline configuration
MathJax.Hub.Config({
    tex2jax: {
        inlineMath: [
            ['$', '$'],
            ['\\(', '\\)']
        ]
    }
});

// Trumbowyg initialization with MathLive button
$('#editor')
.trumbowyg({
    btns: [
        'viewHTML',
        'mathlive'
    ]
});

Setup

In head tag


<link rel="stylesheet" href="trumbowyg/dist/ui/trumbowyg.min.css">

<!-- Import MathLive plugin specific stylesheet -->
<link rel="stylesheet" href="trumbowyg/dist/plugins/mathlive/ui/trumbowyg.mathlive.min.css">

At the end of body


<!-- Import jQuery -->
<script src="//ajax.googleapis.com/ajax/libs/jquery/3.3.1/jquery.min.js"></script>
<script>window.jQuery || document.write('<script src="js/vendor/jquery-3.3.1.min.js"><\/script>')</script>

<!-- Import MathLive -->
<script src="//unpkg.com/mathlive"></script>

<!-- Import MathJax -->
<script src="//cdnjs.cloudflare.com/ajax/libs/mathjax/2.7.5/MathJax.js?config=TeX-MML-AM_CHTML"></script>
